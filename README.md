## What is @ProjectPeitho?
@ProjectPeitho is debate app that allows users to recieve a randomized debate topic, position (affirmative or negative), and a few sources that they can use in their argument to simulate a planned and well-thought-out randomized debate practice. It's pivotal, especially for users like me that participate in speech and debate or TSA, where they offer certain programs with randomized debate topics and time-limits for prep and actual speaking time. It allows the user to branch their scope and understanding of real-world problems today, whether using the app specifically for competitive or even just casual purposes. The app also includes a note-taking section that allows users to view and take notes that they can, again, use in their debate (if looking at the scenario where prep time is needed), or even just jot down quick-thinking ideas they want to implement later on. The app also includes a intuitive timer that shuffles in increments based on the users preferred time.

## Features
- Cute, easy to follow and minimalistic feel that's less daunting, yet still straight to the point.
- Features AI Generated unique, thought-provoking topics that pull real-world themes across tech, ethics, education, science and culture; and never repeats topics to maintain integrity.
- Automatic assignment of position (Affirmative or Negative) whenever a topic is shuffle so debaters have a less-biased perspective on topics.
- Includes relevant sources and facts that provide talking points for the debater to refer during their arguments. Allows for a more concise, quick, and prepared-feel to the debate at hand. 
- Preset 5 minute debate timer with play/pause controls that allow users to increase/decrease time in 1-minute increments (even mid-countdown). 
- A live notepad feature that allows users to jot down quick-minute thoughts before or during a debate with live saving modes. Notes, using localStorage survive page refreshes and browser closes. All notes can be viewed and deleted.

## Want to run locally?
Here's how you can run the server locally:
1. Clone the repo: `git clone <https://github.com/zaishman/debateapp>`
2. Run `npm install` in the terminal
3. Create a `.env` file that includes:
- ANTHROPIC_API_KEY= your anthropic key (visit [claude](https://platform.claude.com/) -> create account -> get free API key)
4. Run `node server.js`
5. Open `http://localhost:3000`

And there you go! Have fun with it!
